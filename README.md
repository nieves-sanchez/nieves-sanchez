## 🧠 Sobre mí

Soy analista de datos junior con orientación técnica, especializada en trabajar todo el ciclo del dato: desde la ingesta y transformación hasta el análisis y la obtención de insights para negocio.

Trabajo principalmente con **Python y SQL**, desarrollando pipelines ETL, modelando bases de datos relacionales y realizando análisis exploratorio (EDA) para entender los datos y extraer valor.

También tengo experiencia en **extracción de datos desde APIs y web scraping (BeautifulSoup y Selenium)**, lo que me permite trabajar con distintas fuentes de información.

Provengo de entornos orientados a negocio (calidad, métricas y operaciones), lo que me permite traducir necesidades reales en análisis útiles y conectar la parte técnica con el contexto empresarial.
